Мои расходы отслеживает ваши расходы и доходы прямо на вашем смартфоне и планшете.

- До 5 счетов с поддержкой <b>операций</b> (поддержка пересчета курсов валют)
- Неограниченное количество <b>шаблонов</b> и до трех планов для <b>повторяющихся</b> транзакций
- <b>Экспорт</b> в QIF и CSV. <b>Импорт</b> из QIF
- Данные могут быть защищены <b>паролем</b> или <b>экраном блокировки</b> устройства.
- <b>Тема</b> (темная и светлая) и <b>размер шрифта</b>могут быть настроены
- <b>Согласование</b> : легко сравнивайте статус транзакции с выписками из своего банка
- <b>Виджет на</b> главном экране и <b>ярлыки</b> для быстрого ввода данных

Получение дополнительных возможностей при покупке одного из следующих лицензионных ключей:

<b>Contrib-ключ</b>

- <b>Без рекламы</b>
- Неограниченное количество <b>аккаунтов</b>
- Неограниченное количество планов для <b>повторяющихся</b> транзакций
- <b>Круговая диаграмма</b> : визуализация статистического распределения транзакций по категориям за разные периоды (годы, месяцы, недели, дни)
- <b>Разделенная транзакция</b> : одна транзакция может быть разделена на несколько частей (категории и переводы).
- <b>Распечатать</b> список транзакций в файл PDF
- <b>Добавление изображений</b>: делайте снимки с камеры или выберите из галереи

<b>Расширенный ключ</b>

- Все возможности Contrib-ключа
- <b>Облачная синхронизация</b> : данные могут быть синхронизированы между несколькими устройствами. (Webdav, Dropbox)
- Ежедневное <b>автоматическое резервное копирование</b> с дополнительной загрузкой в облачное хранилище
- Сложный, настраиваемый <b>импорт CSV</b> (приложения для работы с электронными таблицами)

<b>Professional-ключ</b>

- Все возможности Extended-ключа
- <b>Составление бюджета</b> : сравните свои расходы с лимитами, установленными вами для каждой категории на ежедневной, еженедельной, ежемесячной или ежегодной основе.
- <b>История</b> (Cash flow)
- <b>Сканирование квитанций</b> : автоматическое извлечение суммы, даты и получателя
- <b>Веб-интерфейс пользователя</b>
- Неограниченное количество <b>шаблонов и планов для разделенных транзакций</b>
- <b>E-mail поддержка</b> с обязательным ответом в течении 2 рабочих дней
- Право голоса на <b>дорожной карте разработки</b>

<i>Мои расходы</i> требует следующих <u>разрешений</u>:

- <i>Календарь</i>: Планы будущих и повторяющихся транзакций хранятся либо в специальном локальном календаре, либо в любом другом календаре, который вы назначаете.
- <i>Интернет</i>: Проверка лицензионного ключа, Облачная синхронизация.
- <i>Счета</i>: Синхронизация между устройствами через облачные сервисы.
